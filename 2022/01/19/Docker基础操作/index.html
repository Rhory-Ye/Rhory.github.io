Docker安装Mysql

1.docker安装mysql：sudo docker pull mysql:5.7

2.docker进行mysql文件挂载和配置

参数说明：
--name 指定容器名字
-p 3306:3306 是将容器的3306端口映射到主机的3306端口
-v 是将对应的配置、数据、日志文件夹挂载到Linux主机上。
-e 是配置mysql相关参数的，MYSQL_ROOT_PASSWORD初始化mysql root用户的密码。

修改mysql的配置文件，将字符集的编码格式改为UTF-8

进入mysql容器：docker exec -it mysql /bin/bash

查看当前启动的docker容器： docker ps

重新启动mysql容器：docker restart mysql

设置开机启动docker：sudo systemctl enable docker

设置mysql在docker启动后自启动：sudo docker update mysql --restart=always
